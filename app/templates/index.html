Welcome to use pyrepo

About pyrepo

This is a tool use to analyse datas from git-repositories. It is a local tool developed using python and flask.

How to use

1. You need to copy the .git directory in your git repository to the [path_to_pyrepo]/app/local/projects directory.

2. Go to the root directory of pyrepo and run the script run.py

3. Open your web browser and visit localhost:9999

4. Enjoy it ^_^!
